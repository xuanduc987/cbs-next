# CBS next
> Next-gen Conference room Booking Service

Written in TypeScript, using Angular2 and Firebase, CBS next is your best
choice to book conference room

## Developing

```shell
git clone https://github.com/xuanduc987/cbs-next.git
cd cbs-next/
npm install
npm start
```
